# Gulp Boilerplate for Landing page

### 1. How to run?

###### 1. Setup project
```
yarn
```

###### 2. Run gulp
```
gulp
```

###### 3. Build
```
gulp build
```

#### 2. How to on/off browser sync?
Stop scroll, click, form action in all devices

```
http://localhost:8080/sync-options
```
Choose options what you want to control.
